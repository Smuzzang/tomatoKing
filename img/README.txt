[ 플레이어 캐릭터 아바타 이미지 ]

이 폴더에 아래 두 파일을 넣으면 게임 우측 점수판에 캐릭터가 표시됩니다.

  avatar-me.png   ← 내 자리(우측 하단) 캐릭터
  avatar-opp.png  ← 상대 자리(우측 상단) 캐릭터

권장 사양
  - 정사각형(예: 256x256, 512x512). 가로세로 1:1이 가장 깔끔.
  - 얼굴/상반신이 위쪽에 오게(아바타는 위쪽 기준으로 잘려 보임).
  - PNG(투명 배경 가능) 또는 JPG. 파일명은 위와 정확히 일치.

파일이 없으면 아바타는 자동으로 숨겨지고 기존(텍스트만) 모습으로 표시됩니다.
다른 파일명을 쓰고 싶으면 index.html의 img src 경로를 바꿔주세요.
